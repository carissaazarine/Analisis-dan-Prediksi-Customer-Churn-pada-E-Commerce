# Analisis-dan-Prediksi-Customer-Churn-pada-E-Commerce
Bertujuan untuk memprediksi customer churn pada data e-commerce menggunakan beberapa algoritma machine learning. Dilakukan proses EDA, preprocessing data, serta perbandingan performa model Logistic Regression, KNN, dan Naive Bayes dalam mengklasifikasikan pelanggan yang churn dan tidak churn.
